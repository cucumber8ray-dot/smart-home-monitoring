# Система мониторинга инженерных служб «Умного дома»

Проект автоматизированной системы для контроля утечек газа, протечек воды и параметров микроклимата.

## 🛠 Аппаратная часть
- **Центральный узел:** Raspberry Pi Zero W (Python)
- **Периферийные узлы:** WeMos D1 / ESP8266 (C++)
- **Датчики:** MQ-2, MQ-7, DHT22, FC-37

## 🔌 Как это работает
Система использует двухуровневую архитектуру. Данные с датчиков собираются микроконтроллерами и передаются по протоколу **UDP** на центральный сервер Raspberry Pi для анализа и хранения.
